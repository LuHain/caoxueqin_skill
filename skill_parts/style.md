# Scholarly Style Guide

## Default register

Unless the user's style profile (from style_extractor) overrides, use:
- Academic prose, clear and precise
- Moderate hedging on contested claims
- Chinese characters inline with Pinyin romanization on first appearance
- Footnote-style citation references (can be parenthetical if user prefers)
- Hawkes/Minford translation titles as English reference: *The Story of the Stone*
- Yang/Yang translation as alternative: *A Dream of Red Mansions*

## Chinese text conventions

- Traditional characters (繁體) for all citations
- Simplified (简体) only if user is working in PRC scholarly context
- Pinyin with tone marks for all romanized Chinese
- Wade-Giles only if user's existing work uses it

## Structural conventions for outputs

**Chapter note**:
```
Chapter [N]: [title in Chinese] ([Pinyin])
Edition: [manuscript]
---
[Body: 2–5 paragraphs covering textual, contextual, interpretive dimensions]
[Notes: numbered, at end]
```

**Draft section**:
```
[Section heading]
[Body prose]
[Transition to next section if applicable]
[Notes block]
```

## What to avoid

- Anachronistic psychological language (avoid "trauma," "gaslighting," etc.)
- Overclaiming: "Cao Xueqin intended..." → prefer "the text suggests..." 
- Mixing editions without flagging the switch
- Treating Gao E chapters as Cao Xueqin's without qualification
